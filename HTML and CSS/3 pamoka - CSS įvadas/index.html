CSS - cascadian style sheets - pakopiniai stiliaus šablonai

Uždėti HTML elementui stilius galima keliais 2 būdais
  • Naudojant CSS (pakopiniai stiliaus šablonai)
  • Naudojant style attributą
Stiliai rašomi naudojant CSS todėl:
  • Kad nesudibliuotų kodas(stiliaus aprašai)
  • Jog būtų atskirtas turinys nuo stilių
  • Jog stiliaus šablonai galėtų būt grupuojami ir struktūrizuojami
  • Kad būtų galima įgalinti stiliaus šablonų prioritetus

CSS prioritetai

CSS prioritetai, tai stiliaus savybių perrašymas pagal prioritetų taisykles

Prioritetų taisyklės

  1. Selektoriaus tipas

    Jeigu selektorių tipai yra vienodo prioriteto, tuomet yra vertinamas apibrėžtumas.
  2. Apibrėžtumas - tai papildomas elementų apibūdinimas.
    Jeigu apibėžtumas yra lygaus prioriteto, tuomet vertinama eiliškumas
  3. Eiliškumas - kuo vėliau aprašyta, tuo didesnis prioritetas.

Box-model

Kiekvienas HTML elementas yra stačiakampė dežutė, net jeigu atrodo kitaip.

Kiekvienas HTML elementas turi tokias sekcijas (vardinama iš vidaus į išorę)
  • content - turinys
  • padding - kamšalas
  • border - sienelės
  • margin - paraštės

Padding, border ir margin sekcijos gali būt nustatomos keturiomis kryptymis: ↑ ↓ → ←

Kuomet aprašome elemento padding arba margin CSS savybes, galimi tokie trumpiniai pagal argumentus:

  • a - dydis a į visas puses
  • a b - dydis a vertikaliai, o b horizontaliai
  • a b c - dydis a į viršų, dydis b - į šonus, o c - į apačią
  • a b c d - dydis a į viršų, dydis b į dešinę, c - į apačią, d - į kairę
Pavyzdžiai
  • margin: 20px 5px; - ↑ 20px || ↓ 20px || ← 5px || → 5px
  • margin: 10px; - ↑ 10px || ↓ 10px || ← 10px || → 10px
  • padding: 10px 0 5px; - ↑ 10px || ↓ 5px || ← 0 || → 0
  • padding: 10px 2px 5px 7px; - ↑ 10px || ↓ 5px || ← 7px || → 2px
