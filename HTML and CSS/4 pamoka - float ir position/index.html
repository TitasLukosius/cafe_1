Float

CSS float savybį gali slinkti į vieną iš dviejų pusių: kairę, arba dešinę.
Po slenkančiais(float) elementais, kiti display block elementai, jų tarsi nepripažįsta-nemato, todėl slenekantieji (float) elementai įsiveržia į po apačia esančių display: block elementų kontūrus.
Visgi, po slenkančiais(float) elementais aprašytuose display: block elementuose esantis tekstas, nepersidengia su slenkančiais elementais, o aplink juos "apteka".
