Display - flex

Display flex tipas nurodo tėviniam elementuis kaip išdėstyti savo pirmo lygio vaikus. Standartiškai jis išdėsto elementus iš kairės į dešinę ir bando sutalpinti į vieną eilutę. Jeigu pirmo lygio vaikai į vieną eilutę netelpa, tuomet bandoma juos suspausti.

Tėviniam elementui nustačius papildomasis savybes, gali nustatyti kaip bus išdėstyti vaikai.
